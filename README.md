# 📈 Stock Market Forecasting using LSTM & GRU

This project aims to predict stock market prices using deep learning models like LSTM and GRU.

## 🚀 Features
- 📊 Historical stock data using yfinance
- 🔮 Next-day price prediction
- 📅 Next-week forecasting
- 📈 Trend prediction (Up/Down)
- 🧠 LSTM vs GRU comparison
- 🌐 Streamlit dashboard for visualization

## 🛠️ Tech Stack
- Python
- TensorFlow / Keras
- Pandas, NumPy
- yfinance
- Streamlit
